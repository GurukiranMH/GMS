ABOUT PROJECT

PROBLEM

With the increase in the number of industries in the urban area, the disposal of solid waste is really becoming a big problem, and solid waste includes paper, cardboard, plastic, metal, glass and others. The common way of managing waste is burning waste.
The present way of separating waste/garbage is the hand-picking method

SOLUTION

Building a deep learning model that can detect and classify types of garbage. A web application is integrated, from where we can monitor a garbage likepaper, cardboard, plastic, metal, glass, others and see the analyzed results on UI.
